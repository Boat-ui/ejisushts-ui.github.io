Ejisu Senior High Technical School - Professional website (v2)
Design notes:
- Uses school colors: white and pink (primary accent).
- Clean, airy, professional layout suitable for selling.
- Badge "Noble Warriors" placed under the motto for prominence.
- Minimal, spaced layout with modern typography and CTAs.

Files included:
- index.html, news.html, gallery.html, contact.html, style.css
- assets/ (12 placeholders + news images)

How to use:
1. Replace assets/placeholder*.jpg with real images (same filenames).
2. Replace the logo placeholder by placing a logo file and updating the <div class="logo"> or replacing with an <img> tag.
3. Open index.html in a browser to preview.
4. If you want additional features (animation, CMS, WordPress conversion, or export as zip with real images), send the images or say "Convert to WordPress".
